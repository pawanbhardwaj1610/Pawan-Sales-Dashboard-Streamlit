# Sales Data Analysis and Forecasting Dashboard

This project analyzes sales data and predicts future sales using machine learning and interactive dashboards.

## Features

- Data Cleaning and Processing
- Exploratory Data Analysis
- Top Selling Product Analysis
- Region-wise Sales Heatmap
- Sales Prediction Model
- 90-Day Sales Forecast
- Interactive Dashboard

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- Streamlit
- Prophet

## Dashboard Preview

Shows business KPIs like:

- Total Revenue
- Total Profit
- Profit Margin
- Monthly Sales Trends

## Run the Project

```bash
pip install -r requirements.txt
streamlit run dashboard.py
