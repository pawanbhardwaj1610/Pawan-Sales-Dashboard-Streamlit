# 🚀 Sales Data Analysis and Forecasting Dashboard

## 📖 Problem Statement

Businesses generate large amounts of sales data, but without proper analysis, it is difficult to extract meaningful insights.
The objective of this project is to analyze sales data, visualize key performance metrics, and forecast future sales to support better business decision-making.

---

## 🛠 Models & Tools Used

### 🔹 Data Analysis & Visualization

* Pandas for data manipulation
* Matplotlib / Plotly for visualization

### 🔹 Forecasting Model

* Linear Regression
  Used to predict future sales trends based on historical data.

### 🔹 Dashboard

* Streamlit
  Used to build an interactive and user-friendly dashboard interface.

---

## 📂 Dataset Description

* Sales dataset containing transactional data
* Key features include:

  * Order Date
  * Sales Amount
  * Product Category
  * Region / Customer details
* Data used for both analysis and forecasting

---

## 🔍 Approach

1. Loaded dataset using Pandas
2. Performed data cleaning
3. Fixed date format issues
4. Converted `order_date` into datetime format
5. Conducted Exploratory Data Analysis (EDA)
6. Created KPI metrics
7. Built visualizations
8. Applied Linear Regression for forecasting
9. Developed Streamlit dashboard
10. Designed UI using CSS (Glassmorphism)

---

## 📊 Dashboard Features

* KPI Metrics (Total Sales, Growth, etc.)
* Sales Trends Over Time
* Category-wise Analysis
* Region-wise Insights
* Sales Forecasting
